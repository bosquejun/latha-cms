Rich-text (Lexical) fields now open in a full-screen, keyboard-aware editor on
phones. The inline field collapses to a tappable trigger showing a text preview;
tapping opens a full-screen modal with the editing surface scrollable above a
formatting toolbar pinned to the bottom (above the on-screen keyboard, sized to
the visual viewport), mirroring a mobile social-composer experience. The block
"text style" controls (paragraph, headings, blockquote) now live in their own
toolbar dropdown whose trigger reflects the current block type, instead of being
nested inside the overflow "more" menu. Desktop layout is unchanged.
